# Challenge 05 - Отчет об ошибке

### 1. Что подсказало решение в тексте ошибки?
Сначала программа упала с `SyntaxError: Missing parentheses in call to 'print'`, что указывает на использование синтаксиса Python 2. После добавления скобок возникла ошибка `FileNotFoundError`, указывающая на отсутствие файла с именем `mifile.txt`.

### 2. Как была решена проблема?
1. Строка `print 'i have created'` была исправлена на валидный синтаксис Python 3 с использованием круглых скобок: `print('i have created')`.
2. Название открываемого файла в функции `open()` было исправлено с ошибочного `mifile.txt` на реальное имя файла, присутствующего в каталоге — `myfile.txt`.

### 3. Чему я научился?
Я вспомнил ключевые синтаксические различия между версиями Python 2 и Python 3 в контексте функции вывода текста, а также закрепил навыки отладки ошибок ввода-вывода (I/O).
