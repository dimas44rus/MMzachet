# Challenge 10 - Отчет об ошибке

### Что делает функция в этом задании?
В данном файле функция не объявляется напрямую, а вызывается из внешнего изолированного модуля `fizzbuzz.py`, где инкапсулирован весь алгоритмический цикл проверки кратности.

### 1. Что подсказало решение в тексте ошибки?
Интерпретатор выдал ошибку `TypeError: 'module' object is not callable`. Это означает, что программа пытается вызвать целый модуль (файл) как функцию.

### 2. Как была решена проблема?
Вместо обычного импорта модуля `import fizzbuzz` был применен синтаксис `from fizzbuzz import fizzbuzz`. Это позволило импортировать из файла одноименную функцию напрямую в текущее пространство имен, благодаря чему вызов `fizzbuzz(100)` стал синтаксически корректным.

### 3. Чему я научился?
Я повторил разницу между импортом всего модуля целиком и импортом отдельных его функций через конструкцию `from ... import ...`.
