# Challenge 02 - Отчет об ошибке

### 1. Что подсказало решение в тексте ошибки?
При запуске программа выдавала ошибку `TypeError: fizzbuzz() missing 1 required positional argument: 'max_num'`. Это прямо указало на то, что функция вызвана без обязательного параметра.

### 2. Как была решена проблема?
В блоке `if __name__=='__main__':` вызов функции был изменен с пустого `fizzbuzz()` на `fizzbuzz(100)`. Значение `100` выбрано потому, что функция `range(1, 100)` в Python не включает верхнюю границу, а значит, цикл отработает ровно до 99 включительно, как того требует условие задачи.

### 3. Чему я научился?
Я повторил правила передачи аргументов в функции, а также вспомнил особенность работы функции `range()`, которая не включает конечное значение.
