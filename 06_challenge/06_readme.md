# Challenge 06 - Отчет об ошибке

### Что делает функция в этом задании?
Функция импортирует конфигурационный модуль `conf.py`, извлекает оттуда настройки делителей (числа 3 и 5) и применяет их в цикле фильтрации последовательности чисел.

### 1. Что подсказало решение в тексте ошибки?
Интерпретатор выдал ошибку `AttributeError: module 'conf' has no attribute 'num'`, указав на строку 20. Это значит, что в импортированном модуле `conf` нет переменной с именем `num`.

### 2. Как была решена проблема?
При изучении файла конфигурации `conf.py` обнаружилось, что целевая переменная называется `num2`, а не `num`. В основном файле опечатка была исправлена: строка `num2 = conf.num` была заменена на `num2 = conf.num2`.

### 3. Чему я научился?
Я закрепил навыки работы с модулями в Python и научился отлаживать ошибки отсутствия атрибутов (`AttributeError`) при импорте внешних конфигурационных файлов.
