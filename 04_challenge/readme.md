# Challenge 04 - Отчет об ошибке

### 1. Что подсказало решение в тексте ошибки?
Интерпретатор выдал ошибку `IndentationError: expected an indented block` на строке, следующей за условием `if`.

### 2. Как была решена проблема?
В Python блоки кода выделяются отступами, а не фигурными скобками. Тело условия `if i%num1==0 and i%num2==0:` не имело нужного смещения вправо. Проблема решена добавлением стандартного отступа (4 пробела) перед функцией `print()`.

### 3. Чему я научился?
Я закрепил понимание важности правильных отступов в Python, так как они напрямую влияют на синтаксическую корректность структуры программы.
