# Challenge 08 - Отчет об ошибке

### 1. Что подсказало решение в тексте ошибки?
Сначала в коде присутствовала ошибка `NameError: name 'fizzbuzz' is not defined`. После обращения через экземпляр класса возникла стандартная для ООП ошибка `TypeError`, указывающая на неверное количество переданных аргументов.

### 2. Как была решена проблема?
Функция `fizzbuzz` была обернута в класс `Fizz_Buzz`, став его методом. В Python первым аргументом любого метода экземпляра класса должен быть `self`. В объявлении метода `def fizzbuzz(max_num):` этот параметр отсутствовал. Ошибка исправлена добавлением `self` в качестве первого параметра: `def fizzbuzz(self, max_num):`.

### 3. Чему я научился?
Я вспомнил основы ООП в Python: методы класса неявно передают ссылку на сам объект в качестве первого аргумента, поэтому ключевое слово `self` строго обязательно при объявлении таких методов.
