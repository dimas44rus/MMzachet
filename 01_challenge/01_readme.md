# Challenge 01 - Отчет об ошибке

### 1. Что подсказало решение в тексте ошибки?
Интерпретатор Python выдал `SyntaxError: invalid syntax` и указал стрелочкой на строку 27 в конструкции `elif i%num1=0:`.

### 2. Как была решена проблема?
Внутри оператора ветвления `elif` использовался одиночный знак равенства `=`, который в Python служит оператором присваивания. Для проверки равенства (сравнения чисел) необходимо использовать двойное равенство `==`. Ошибка исправлена путем замены `=` на `==`.

### 3. Чему я научился?
Я вспомнил базовое отличие синтаксиса Python: `=` — это присваивание значения переменной, а `==` — это сравнение двух выражений.
